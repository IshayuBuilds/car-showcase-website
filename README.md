# 🚗 Car Showcase Website

A modern car showcase website built using HTML and CSS featuring responsive card layouts and interactive UI elements.

## 🔥 Features
- Responsive card layout using Flexbox
- Interactive hover effects
- Clean and modern UI design
- Navbar, Hero section, and Footer

## 🛠️ Tech Used
- HTML5
- CSS3 (Flexbox)
